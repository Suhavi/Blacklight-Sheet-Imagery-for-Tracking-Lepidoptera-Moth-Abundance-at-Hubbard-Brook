# Blacklight-Sheet-Imagery-for-Tracking-Lepidoptera-Moth-Abundance-at-Hubbard-Brook
This repository uses blacklight-sheet invertebrate imagery from the Hubbard Brook Experimental Forest to estimate Lepidoptera (moth) abundance indices via an object-detection pipeline (inference + NMS), aggregation to site-level time series, and trend visualization/validation.

# PIPELINE

Images --> Inferencing --> Count --> Trends
